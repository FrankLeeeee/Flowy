Get Started
Visit GitHub

1. Start the hub

Install `@frankleeeee/flowy`, launch the web app, and copy the runner registration secret.

2. Connect a runner

Install `@frankleeeee/flowy-runner` on any machine that has Claude Code, Codex, or Cursor Agent available.

3. Dispatch work

Create a project, add a task, choose a runner, and pass through only the harness settings your CLI needs.
